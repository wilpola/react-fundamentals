# What is react?

React (https://reactjs.org) is javascript library for developing user interfaces. React is developed by Facebook and it is open source library (MIT license). React is component based and the components are reusable. After you have learned React, you can also develop mobile applications using React Native.

Create React App (https://reactjs.org/docs/create-a-new-react-app.html) is the best way to build a new React app. It provides all neessary tools for you to start efficient React development. To use Create React App, you have to install Node Version >= 8.10 and Npm version > 5.6.

The programming language of React library is Javascript therefore the basic knowledge of Javascript programming language is needed in this course. You can also use TypeScript, but in this course we are using the Javascript.
